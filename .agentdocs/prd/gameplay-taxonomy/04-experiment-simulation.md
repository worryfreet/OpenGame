# 实验模拟与变量调参类

## 适合什么知识

实验模拟与变量调参类玩法解决“改变一个条件，结果怎么变”。它适合科学实验、AI 参数、模型训练、Prompt 迭代、物理/化学变量、生态系统和复杂系统理解。

不适合：没有变量关系的纯事实记忆。若知识点只是术语背诵，应使用记忆或匹配玩法。

## 变量调节

### 滑杆调参

核心动作：移动滑杆改变变量。

交互变体：

- 权重滑杆。
- 温度 / 速度 / 力大小。
- 生成风格强度。
- 难度参数。

反馈方式：实时曲线、结果预览、阈值警报。

适合：物理量、神经网络权重、Prompt 风格、函数参数。

### 开关组合

核心动作：打开或关闭条件，观察组合结果。

交互变体：

- 实验条件开关。
- 逻辑门。
- 系统权限开关。
- 生物环境条件。

反馈方式：条件组合表、结果状态灯、冲突提示。

适合：科学实验、编程逻辑、Agent 工具授权、生态因素。

### 参数面板

核心动作：填写或选择多项参数后运行。

交互变体：

- 表单配置。
- 实验仪器面板。
- Prompt 参数卡。
- 生成器配置。

反馈方式：运行日志、输出质量评分、参数建议。

适合：AI 工具、科学探究、项目配置、Skill 输入。

## 对比实验

### A/B 对比

核心动作：运行两个版本并比较结果。

交互变体：

- 两版 Prompt 输出。
- 两组实验数据。
- 两种策略收益。
- 两段代码效果。

反馈方式：差异高亮、指标对比、推荐选择原因。

适合：Prompt 迭代、科学变量控制、写作修改、产品方案比较。

### 控制变量实验

核心动作：只改变一个变量，观察结果。

交互变体：

- 固定其他条件。
- 拖入唯一变量卡。
- 自动拦截多变量改动。
- 生成实验记录。

反馈方式：提示变量污染、结果可信度、实验结论卡。

适合：科学方法、物理、化学、生物、数据分析。

### 误差实验

核心动作：观察测量误差或训练误差如何影响结果。

交互变体：

- 添加噪声。
- 调整误差阈值。
- 比较预测与真实。
- 误差反推调参。

反馈方式：误差曲线、残差可视化、校准建议。

适合：数学统计、模型训练、物理测量、AI 评价。

## 迭代优化

### 多轮迭代

核心动作：根据反馈修改输入，再运行。

交互变体：

- Prompt V1/V2/V3。
- 作文草稿修改。
- 模型训练轮次。
- 游戏需求迭代。

反馈方式：版本记录、质量曲线、修改影响说明。

适合：Prompt、写作、Vibe Coding、工程优化。

### 调参挑战

核心动作：在目标范围内调出合格结果。

交互变体：

- 让曲线进入目标区。
- 让机器人稳定运行。
- 让输出满足格式。
- 让化学反应达到平衡。

反馈方式：目标区、实时评分、过调警告。

适合：函数、物理、化学、AI 参数、控制系统。

## 沙盒模拟

### 生态 / 城市沙盒

核心动作：设置多个因素，观察系统长期变化。

交互变体：

- 生态种群。
- 城市交通。
- 小镇资源。
- 数据中心负载。

反馈方式：多轮演化、折线图、系统崩溃或稳定状态。

适合：生物、地理、社会科学、系统思维。

### 物理沙盒

核心动作：搭建物理场景并观察运动。

交互变体：

- 力和运动。
- 电路搭建。
- 光路反射。
- 抛物线。

反馈方式：轨迹、数值读数、失败原因。

适合：物理、工程、数学函数。

### AI 行为沙盒

核心动作：设置输入、规则、知识和工具，看 AI 行为变化。

交互变体：

- Prompt 沙盒。
- Agent 工具授权。
- RAG 来源切换。
- 安全规则开关。

反馈方式：输出对比、风险提示、行为日志。

适合：AI、Agent、责任使用、项目实操。
